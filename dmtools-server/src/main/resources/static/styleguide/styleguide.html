Welcome to the DMTools Style Guide

This style guide is a living document that showcases the UI components, design patterns, and visual guidelines for the DMTools application. Use the navigation above to explore different categories of components.

The goal is to create a consistent and high-quality user experience across all parts of DMTools. This guide helps developers and designers to:

  • Understand the available UI building blocks.
  • Reuse components to ensure consistency.
  • Test components in isolation.
  • Quickly reference design specifications.

Dependencies

The DMTools component library requires the following external dependencies:

  • Font Awesome 6.0.0 - Used for icons throughout the application. Include via CDN:
    <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.0.0/css/all.min.css">
  • CSS Custom Properties - All components use CSS variables for theming and consistency
  • theme.js - JavaScript for theme switching functionality
