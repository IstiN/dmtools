Atoms

The smallest, most basic building blocks of the UI. These examples should render using the styles from ../css/styles.css.

Buttons

Core button style is .btn. Modifiers provide different appearances and states.

Base Button (.btn)

<button class="btn">Default Button</button>

Primary Button (.btn-primary)

<button class="btn btn-primary">Primary Action</button>

Secondary Button (.btn-secondary)

<button class="btn btn-secondary">Secondary Action</button>

Outline Button (.btn-outline)

<button class="btn btn-outline">Outline Button</button>

Note: .btn-outline may require a specific background in some themes to be visible, as it might use light colors.

Small Button (.btn-small)

<button class="btn btn-primary btn-small">Small Primary</button>
<button class="btn btn-secondary btn-small">Small Secondary</button>

Icon Button (.btn-icon)

<button class="btn-icon" aria-label="Description">
    <!-- SVG or Icon Font -->
</button>

Form Inputs

Common form elements, typically wrapped in a .form-group.

Text Input

<div class="form-group">
    <label for="some-id">Text Label</label>
    <input type="text" id="some-id" placeholder="Enter text...">
</div>

Password Input

<div class="form-group">
    <label for="some-pass-id">Password Label</label>
    <input type="password" id="some-pass-id" placeholder="Enter password...">
</div>

Number Input

<div class="form-group">
    <label for="some-number-id">Number Label</label>
    <input type="number" id="some-number-id" placeholder="Enter a number...">
</div>

Text Area

<div class="form-group">
    <label for="some-textarea-id">Textarea Label</label>
    <textarea id="some-textarea-id" placeholder="Enter details..."></textarea>
</div>

Select Dropdown

<div class="form-group">
    <label for="some-select-id">Select Label</label>
    <select id="some-select-id">
        <option>Option 1</option>
        <option>Option 2</option>
    </select>
</div>

Checkbox (styling may vary, using browser default if not specified in styles.css)


<div class="form-group">
    <label><input type="checkbox"> Option 1</label>
    <label><input type="checkbox" checked> Option 2</label>
</div>

Radio Buttons (styling may vary, using browser default if not specified in styles.css)


<div class="form-group">
    <label><input type="radio" name="radio-group"> Option A</label>
    <label><input type="radio" name="radio-group" checked> Option B</label>
</div>

Tags (.tag)

Tags are used for keywords or labels. They use the .tag class.

Default Tag Another Tag 
<div class="tag-container">
    <span class="tag">Tag Text</span>
</div>

Status Dots (.status-dot)

Status dots are small indicators, often used with text.

Successful/Online
Warning
Error/Failed
Offline/Muted
<span class="status-dot"></span>
<span class="status-dot warning"></span>
<span class="status-dot error"></span>
<span class="status-dot offline"></span>

View All Link (.view-all)

A styled link component with arrow icon used for "view all" or "see more" actions.

View All

See More Items 
<a href="#" class="view-all">
    View All <i class="fas fa-chevron-right"></i>
</a>

Run Button (.run-button)

A specialized button for triggering agent runs or executions.

<button class="run-button">
    <i class="fas fa-play"></i> Run
</button>
