Molecules

Simple groups of UI elements functioning together as a unit.

Theme Switch

The theme switch component allows users to toggle between light and dark modes. It uses the .theme-switch class and typically contains an icon and optional text. Its functionality is usually driven by JavaScript (e.g. theme.js).

<button class="theme-switch" aria-label="Toggle theme">
    <!-- SVG Icon for theme (sun/moon) -->
    <span class="theme-text">Toggle Theme</span> <!-- Optional text, often hidden on smaller screens -->
</button>

Card (.card)

Cards are used to group related information. The base .card style provides background, border, shadow, and rounded corners. Content can vary.

Card Title

This is some content within a basic card. Cards can contain various elements like text, images, and buttons.

<div class="card">
    <!-- Card content goes here -->
    <div style="padding: 1rem;">
        <h4>Card Title</h4>
        <p>Card content...</p>
    </div>
</div>

Search Form

A common molecule for search functionality, combining an input field and a button.

<form class="search-form">
    <input type="search" placeholder="Search..." aria-label="Search">
    <button type="submit" class="btn">Search</button>
</form>

Chat Message (.chat-message)

Displays a single message in a chat interface, with variations for sender.

User Message

This is a message from the user. It's aligned to the right.
<div class="chat-message chat-message--user">User message text</div>

Agent Message

This is a message from the agent. It's aligned to the left and has a different background.
<div class="chat-message chat-message--agent">Agent message text</div>

System Message

This is a system notification or status update within the chat.
<div class="chat-message chat-message--system">System message text</div>

Chat Input Group (.chat-input-group)

A group containing a textarea for message input and action buttons like send or attach.

<div class="chat-input-group">
    <textarea placeholder="Type your message..."></textarea>
    <button class="btn btn-icon" aria-label="Attach file">
        <i class="fas fa-paperclip"></i>
    </button>
    <button class="btn btn-primary">Send</button>
</div>

Action Button Group (.action-button-group)

A simple horizontal group of action buttons, typically used for form submissions or main actions in a section.

<div class="action-button-group">
    <button class="btn btn-secondary">Secondary Action</button>
    <button class="btn btn-primary">Primary Action</button>
</div>
