JS AI Client
Path to the JavaScript AI client script file (alternative to inline script below).
JavaScript AI Client Template:
Your script should export a handleChat function that receives:
• messagesString - JSON string of messages array
• modelToUse - AI model name (e.g., "gpt-4", "gemini-2.0-flash")
• metadataString - JSON string of metadata (temperature, max_tokens, etc.)
• javaClient - Java client instance for HTTP calls and logging

Available javaClient methods:
• javaClient.executePost(url, payload, headers)
• javaClient.jsLogInfo(message), javaClient.jsLogWarn(message), javaClient.jsLogError(message)

Secret injection: Use ${secrets.API_KEY_NAME!''} for FreeMarker template variables.
